## Описание проекта.

Это консольное приложение на Python для поиска фильмов.
Программа позволяет искать фильмы по ключевому слову или по жанру и диапазону годов.

Проект использует: 
MySQL (база фильмов) 
MongoDB (логирование поисковых запросов)

------------------------------------------------------------------------

## Возможности программы.

- Поиск фильмов по ключевому слову.
- Поиск по жанру и диапазону годов.
- Показ результатов по 10 фильмов.
- Логирование поисковых запросов в MongoDB.
- Просмотр популярных поисковых запросов.
- Поддержка нескольких языков:
    - Русский
    - Українська
    - English
    - Deutsch
- Возможность сменить язык во время работы программы.
- Проверка подключение к MySQL и MongoDB.



------------------------------------------------------------------------

## Структура проекта.

- main.py - основная логика программы и меню.
- config.py - ввод данных для подключение к MySQL.
- mysql_connector.py - подключение к MySQL и SQL запросы.
- formatter.py - вывод результатов поиска.
- log_writer.py - запись логов в MongoDB.
- log_stats.py - вывод популярных поисков.
- lang - файлы локализации.

------------------------------------------------------------------------
